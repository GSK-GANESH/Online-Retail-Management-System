# Online Retail Management System

## Project Overview
The Online Retail Management System is a Java-based desktop application designed to manage retail store operations efficiently. It provides functionalities for product management, customer handling, sales tracking, and database connectivity.

## Features
- User Login System
- Product Management
- Add New Products
- Update Product Information
- Sales Management
- Database Connectivity using JDBC
- Dashboard for Monitoring

## Technologies Used
- Java
- JDBC
- MySQL
- Swing (GUI)
- Git & GitHub

## Project Structure
- AddProduct.java
- Dashboard.java
- DBConnection.java
- Login.java

## How to Run
1. Install Java JDK.
2. Configure MySQL database.
3. Update database credentials in `DBConnection.java`.
4. Compile the project:
   ```bash
   javac *.java
   ```
5. Run the application:
   ```bash
   java Login
   ```
